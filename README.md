
# Projeto Geoquiz V1

Um jogo de adivinhação de cidades com base em uma API do Google Street View.



## Funcionalidades

- Exibição de um local aleatório do Street View
- Sistema de pontuação (acertos consecutivos)
- Salvamento de pontuação ao errar, com nome do jogador
- Leaderboard com as 10 melhores pontuações
- Chave da API do Google Maps protegida via variável de ambiente



## Tech Stack

- React 18
- Vite
- Google Maps API


## Variável de âmbiente

Para executar este projeto, você precisará adicionar as seguintes variáveis ​​de ambiente ao seu arquivo .env

`VITE_GOOGLE_MAPS_API_KEY`
